# Project Context

## Project Overview

**Himna Marketing** is a modern, responsive marketing agency website built with Next.js 16 and React 19. The project showcases creative design and advanced frontend interactions, targeting Arabic-speaking clients in the Middle East.

## Project Purpose

Himna Marketing serves as a digital marketing agency's online presence, designed to:

1. **Showcase Services**: Display marketing services including social media management, content creation, advertising campaigns, and consulting
2. **Portfolio Display**: Present past work across video and photography categories
3. **Client Engagement**: Convert visitors into leads through strategic CTAs (Call-to-Actions)
4. **Client Testimonials**: Build trust through real client feedback and success stories
5. **Professional Image**: Establish credibility as a modern, tech-savvy marketing agency

## Business Context

- **Target Audience**: Arabic-speaking businesses and entrepreneurs
- **Geographic Region**: Middle East (RTL language support)
- **Business Model**: Service-based marketing agency
- **Key Services**:
  - Social Media Management
  - Content Creation & Design
  - Advertising Campaigns
  - Marketing Consulting
  - Photography & Videography

## Key Features

1. **Landing Page Sections**:
   - Hero section with prominent CTA
   - About section with company vision
   - Services showcase
   - Statistics/Counters
   - Portfolio gallery with video and photography
   - Client testimonials
   - Partners section
   - Contact/CTA section
   - Footer with social links

2. **Interactive Elements**:
   - Smooth scroll animations
   - Mobile-responsive navigation with hamburger menu
   - Portfolio filtering by type (Video/Photography)
   - Animated counters in statistics
   - YouTube video embeds in portfolio
   - Theme support (light/dark mode ready)

3. **Performance & Quality**:
   - React Compiler enabled for optimized rendering
   - TypeScript for type safety
   - Modern CSS with Tailwind
   - Responsive design principles
   - RTL language support

## Development Status

The project is in **active development** as a complete MVP (Minimum Viable Product) with all core sections implemented and functional.

## Technology Leadership Decision

The decision to use Next.js 16 with the App Router reflects a modern, forward-thinking approach:
- Server Components for better performance
- Built-in optimizations for fonts, images, and meta tags
- Fast refresh during development
- Simplified routing with file-based structure
- React 19 adoption for latest features and performance improvements
- React Compiler for automatic rendering optimizations
